# 🚇 Hyderabad Metro AI Android Application

An intelligent Android application for Hyderabad Metro commuters featuring AI-powered assistance, ticket booking, route planning, multilingual support, and digital ticket management.

---

## 📱 Overview

Hyderabad Metro AI is a modern Android application developed using **Kotlin** and **Jetpack Compose**. It helps users book metro tickets, calculate fares, find routes, manage wallets, and interact with an AI-powered metro assistant.

---

## ✨ Features

### 🎫 Smart Ticket Booking
- Book Single & Return Tickets
- Digital QR Tickets
- Wallet Integration
- Automatic Fare Calculation

### 🤖 AI Metro Assistant
- Ask Metro-related questions
- Route Suggestions
- Fare Information
- Tourist Guidance
- Multilingual Responses

### 🚉 Route Planning
- Source & Destination Selection
- Route Calculation
- Interchange Detection
- Distance & Time Estimation

### 🌐 Multilingual Support
- English
- Telugu
- Hindi

### 🔊 Voice Assistant
- Text-to-Speech Announcements
- Voice Interaction Support

### 💳 Wallet
- Wallet Balance
- Recharge Wallet
- Ticket Payment

# 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/nallajashwanth/Hyderabad-Metro-AI-Android-App.git
```

2. Open in Android Studio

3. Sync Gradle

4. Add your API key

```
OPENROUTER_API_KEY=YOUR_API_KEY
```

5. Run the application
